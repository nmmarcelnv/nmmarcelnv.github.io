Getting Help

Before anything else, it's important to know where to get help on a given topic. The reason is that there is so much that it's impossible to remember all. The good news is that help is just one simple function away: Help. Simply pass the name of the function to help and python will print out everyting you need to know about the function.

help(pd.read_csv)

Exploring Data¶

Back to top¶

pandas has many functions that can be used to gain insight from the data. Some of the functions are below.

df.head(n)                               #First n rows of the DataFrame
df.tail(n)                               #Last n rows of the DataFrame
df.shape()                               #Number of rows and columns
df.info()                                #Index, Datatype and Memoryinformation
df.describe()                            #Summary statistics for numerical columns
df.colname.value_counts(dropna=False)    #Views unique values and counts (method for pandas series)
df.apply(pd.Series.value_counts)         #Unique values and counts for all columns   
df.dtypes                                #datatype of each column   
df.sum()                                 # Sum of values in a data frame
df.min()                                 # Lowest value of a data frame
df.max()                                 # Highest value
df.idxmin()                              # Index of the lowest value
df.idxmax()                              # Index of the highest value
